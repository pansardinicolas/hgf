Instructions

  1. Upload your input sequences

    2. All the input sequences must be in FASTA format (.fasta) (.fas) as in one-letter nucleotide code (A, C, G, T and N (unknown)).
    Select individual files from your file system. Multiple files are allowed you can upload as many as you wish, thereafter, they will be individually processed.
    Or it is possible to paste your imput sequence in the text area, the title is optional, if the field is submitted empty it will be named as the current time and
    date in a string with "'month day''mont'hours':'minutes':'seconds'.fasta" (e.g., 10September16:03:02.fasta).

  2. Run gene marking tool

    3. Push the "run" button and then a page will be displayed with your ID, with the status of your files and a link which can be clicked when the status becomes
    completed. Whenever it is allowed to return to the main page and keep uploading, or even to close the page and return by pasting the ID over the url
    (www.bioinfo.ufpr.br/jobqueue.php?id=XXXXX).

  3. Pick up your results

    4. You can choose to download your result by clicking over the download link or to display on screen by clicking over the fasta file's name.
    If you uploaded more than one input it will load until all of them are processed, result's links will appear upon the conclusion of each individual process.
    Your session remains open until the browser is closed, furthermore, result files will be available for about 24 hours.
