HGF Instructions Output

Output Format

Description

The output conforms to GenBank format (.gbk), which contains three sections:

  • Header:
    • Contains information about organism, publications, accession ID, and others.
  • Features:
    • Contains all sequence annotations as visualized by Artemis®. A list of pre annotation blocks of gene, with the predicted CDS (protein coding sequence),
    its ORF range limits (open reading frame, which means a nucleotide sequence between a start and stop codons, long enough to encode for a typical
    protein) and its color schema, determined depending on the ORF's predictive score, ranging from gray, pink and red (regarding to increasing confidence,
    respectively).
  • Origin:
    • Contains the DNA sequence.
